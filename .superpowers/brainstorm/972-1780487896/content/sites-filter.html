One question

When "BP Sites" is selected — should the main domain column be hidden, or shown alongside the BP columns?

A

Hide main

BP Sites = only the 4 BP domain columns. Main column is hidden. URL: /bp-sites/lucky7even/bp

B

Include main alongside

BP Sites = main column + all 4 BP columns (same as "All" but with a different label). Only differs from All if "All" also adds something.
